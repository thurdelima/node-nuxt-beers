## 🧰 Backend:

Configure connection postgress in .env.example (.env) file.

Run migrations.

```bash
  adonis seed --files database/seeds/CategorySeeder.js
  adonis seed --files database/seeds/BeerSeeder.js
```

And run project.

```bash
  npm run start
```

## 🧪 Tests:

To run tests, just type npm run test.

```bash
  npm run test
```
